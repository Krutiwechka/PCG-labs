Приложение для анализа метаданных графических файлов. Поддерживает форматы: JPG, GIF, TIFF, BMP, PNG, PCX.
Основные функции

    Пакетная обработка до 100 000 файлов

    Извлечение метаданных: размер, разрешение, глубина цвета, сжатие

    Многопоточная обработка для высокой производительности

    Табличный интерфейс

Запуск приложения

pip install -r requirements.txt
python main.py

Отображаемая информация

    Имя файла - название графического файла

    Размер - ширина и высота в пикселях

    Разрешение - DPI (dots per inch)

    Глубина цвета - бит на пиксель

    Сжатие - алгоритм сжатия данных

    Формат - тип графического файла

    Режим - цветовая модель (RGB, Grayscale и т.д.)

Техническая реализация

    Библиотека обработки изображений: Pillow (PIL) - обеспечивает чтение метаданных всех поддерживаемых форматов

    Графический интерфейс: PySide6 - кроссплатформенная библиотека на основе Qt

    Многопоточность: ThreadPoolExecutor - параллельная обработка файлов без блокировки интерфейса

    Архитектура: Модель MVC с разделением интерфейса и логики обработки

Инструкция по использованию

    Выберите папку с графическими файлами через кнопку "Обзор"

    Запустите процесс сканирования кнопкой "Начать сканирование"

    Наблюдайте прогресс обработки в реальном времени

    Анализируйте результаты в табличном представлении

    Для детальной информации выберите конкретный файл в таблице

Особенности реализации

    Обработка ошибок для каждого файла индивидуально

    Автоматическое определение формата файла

    Поддержка EXIF-данных для JPEG и TIFF

    Статистика обработки и отчет об ошибках

    Возможность остановки процесса в любой момент
