requirements: Python, PyQt5

Описание программы

Программа для перевода цветов между моделями RGB, XYZ и HLS. Реализована на Python с использованием библиотеки PyQt5. Для работы требуется Python и установленный пакет PyQt5.

Интерфейс

    Три панели для разных цветовых моделей: RGB (0-255), XYZ (0-100), HLS (H: 0-360, L/S: 0-100)

    Изменение цвета через ползунки, числовые поля или выбор из палитры (кнопка "Выбрать из палитры")

Особенности работы

    Конвертация выполняется по указанным на портале формулам преобразования цветовых моделей

Техническая реализация

    Использованы спинбоксы для ограничения вводимых значений

    Реализована синхронизация между ползунками и числовыми полями
